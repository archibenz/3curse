# Maze Project (DAGA/proj)

GUI‑приложение для генерации и визуализации лабиринта с многопоточной синхронизацией и сравнительными тестами.

## Структура проекта

- `src/maze/` — исходники лабиринта и GUI.
- `scripts/` — вспомогательные скрипты (установка/сборка, генерация графиков).
- `assets/` — примеры CSV/PNG из тестов.
- `outputs/` — создаётся автоматически для результатов тестов и графиков.
- `artifacts/` — собранные бинарники/приложения (исторические).
- `build/` — директория сборки (создаётся CMake).

## Зависимости

**Обязательные:**
- C++17 компилятор (GCC/Clang/MSVC)
- CMake 3.16+
- Qt 6 (Widgets)

**Опционально:**
- Python 3 + `matplotlib` — для построения графиков производительности (кнопка в GUI).

## Сборка (универсальная: CMake)

```bash
cmake -S . -B build
cmake --build build
```

Запуск:

```bash
./build/maze_gui
```

## Сборка (альтернатива: qmake)

```bash
qmake 4sem.pro
make
```

Запуск (пример):

```bash
./4sem
```

> Примечание: qmake‑сборка — историческая, может требовать ручной настройки путей Qt.

## Быстрый setup

- **Linux/macOS:** `scripts/setup.sh`
- **Windows:** `scripts/setup.bat` (через MSYS2)

## Как пользоваться

1. Запустите GUI.
2. В меню выберите генерацию лабиринта.
3. Кнопка «Графики производительности» создаёт CSV и PNG в папке `outputs/`.
