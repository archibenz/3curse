Maze Web

Построение лабиринтов

Демо-режим показывает генерацию и анимацию пути (данные приходят с Java backend).

Описание

  • Построение: DFS-Backtracking с разбиением на потоки.
  • Поиск пути: BFS, гарантирует кратчайший путь.
  • Синхронизация: локальные и глобальные барьеры.

Графики производительности

Web-интеграция

Backend: Java HttpServer. Frontend: статический HTML/CSS/JS.

Запуск: javac src/Main.java и java Main из каталога web/backend.

Затем открыть http://localhost:8080.
